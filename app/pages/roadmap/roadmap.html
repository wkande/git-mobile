Functionality

Everything in version 1 is read only. Look to version 2 to begin with items such as "create issues" and other write capabilities.


The plan is to try and cram as much functionality into version 1x as we can. Not everything can make v1 and some will move to version 2. With each new alpha release during the test phase you will see new functionality added. Beta releases will not see new functionality and are only for code cleanup/bugs.


As a tester we really value your input. Please use the Issue Tracking on the GitHub public repo for Git-Mobile. The only way to be heard is to speak. We're listening.
